Acerca de

Para este portfolio he decidido ser creativo e implementar un estilo 'Pixel', dándole forma a títulos y botones asi como aplicando distintos efectos de 'glow' haciendo uso de propiedades como box-shadow y jugando con el background-color y el propio color de las letras. Además he aplicado distintas transiciones de flotado (en el Home) haciendo uso de keyframes de forma que la imagen suba y baje en bucle.

En este caso, he implementado un menú desplegable una vez que la página alcanza los 1314px. Este menú lo he implementado haciendo uso de Javascript mediante un simple EvenListner de forma que al hacer click se desplace el menu desde arriba.

La estructura de la página esta realizada con Flex-Box al completo haciendo uso de las propiedades básicas como: Flex-direction, Flex-wrap o justify-content. De esa forma consigo una página responsive muy bien estructurada. Cuando la página cambia a resolución movil, cambia el formato del título y los botones inferiores para que se aprecie mejor el cambio de resolución.
